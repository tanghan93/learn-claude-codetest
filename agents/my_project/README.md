# My Python Project

A simple Python project demonstrating basic structure.

## Files

- **main.py** - Main entry point. Interacts with the user and calls utility functions.
- **utils.py** - Utility functions including `greet(name)` and `add(a, b)`.
- **README.md** - This file, describing the project.

## How to Run

```bash
cd my_project
python main.py
```

## Requirements

- Python 3.6 or higher
